# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its code of conduct. Please read the linked document for the full text of the standards and
expectations that apply to everyone participating in this project's spaces.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the project maintainers through
GitHub — open a private report via the repository **Security** tab, or contact a maintainer directly. Do not
open a public issue for conduct reports.

All reports will be reviewed and handled confidentially.
